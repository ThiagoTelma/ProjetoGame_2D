## 📖 Sobre o Jogo

**Indiano Gomes: Em Busca do Tesouro Perdido** é um jogo de aventura 2D desenvolvido em Java utilizando conceitos de Programação Orientada a Objetos.

O jogador assume o papel de um explorador em busca de um antigo tesouro escondido em uma floresta repleta de mistérios, armadilhas e desafios.

Durante a jornada será necessário:

* 🪙 Coletar moedas espalhadas pelo mapa.
* 🤝 Negociar com personagens da floresta.
* 🔦 Conseguir uma fonte de luz para explorar áreas escuras.
* 🗝️ Encontrar uma chave secreta enterrada.
* ⛏️ Escavar locais suspeitos.
* 🏛️ Acessar um templo antigo.
* 💎 Recuperar o tesouro perdido.
